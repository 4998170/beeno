Beeno by ghelmling

Simple Java Beans mapping for HBase

The native Java API for HBase provides fairly low level access to your data, representing rows essentially as maps of byte arrays. To simplify data access and manipulation, Beeno provides a simple entity layer that maps Java classes -> HBase tables and back. This layer borrows a bit conceptually from Hibernate and JPA, though it's much, much more limited in scope.

Dependencies

  • Apache HBase 0.20.x
  • Apache Hadoop 0.20.x
  • Log4J 1.2.15
  • Google Protocol Buffers 2.0.3
  • ZooKeeper 3.2.1
